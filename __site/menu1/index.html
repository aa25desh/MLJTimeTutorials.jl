Online Download

TSdataset(dataset::Array) function adds new data set from timeseriesclassification.com. to "data" folder for the MLJTime directory. Make sure that your path must be in the MLJTime directory. For example take ACSF1 
TSdataset(["ACSF1"])

With TS Files

TSdataset(filepath::String) fucntion converts .ts files to .csv files & add in same folder so one create julia data. Path to the Dir will be inpute in this case. 
TSdataset(""/Users/abc/MLJtime_pdf_data/GunPoint")
